# MatchPointML

## Sobre
MatchPointML é um projeto de análise de dados e machine learning. Nele foi feito scrapping para a obtenção de dados de todos os jogos da ATP (liga proficional de tênis) desde os anos 60. Em seguida foi feito um tratamento de dados e elaborado um modelo de ML (Manhine Learning). Foi desenvolvido uma plataforma frontend para ter acesso ao modelo e fazer previsões

## Propósito
O objetivo é fornecer uma base reutilizável e extensível para a pesquisa, prototipagem e integração de algoritmos de matching baseados em ML em aplicações como visão computacional, reconstrução 3D, registro de nuvens de pontos e localização.

## Principais funcionalidades
- Pipeline de pré-processamento e normalização de pontos.
- Extrator de características e embeddings para pontos.
- Módulos de correspondência (clássicos e baseados em ML).
- Métricas de avaliação e visualização de resultados.
- Exemplos e notebooks para reprodução de experimentos.

## Tecnologias
- Python, NumPy, PyTorch (ou outra framework ML), Open3D (opcional)
- Scripts para treinamento, inferência e avaliação.

## Como começar (rápido)
1. Clone o repositório.
2. Crie um ambiente virtual e instale as dependências: `pip install -r requirements.txt`
3. Execute um exemplo em `examples/` para ver o fluxo completo.
